# Master Designer V3.0 (React + Tailwind)

This is the new React-based implementation of Master Designer V3.0, featuring a modern, high-performance design.

## Tech Stack
- **Framework**: React 19 (Vite)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

1.  Navigate to the directory:
    ```bash
    cd v3
    ```

2.  Install dependencies (if not already):
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Build for production:
    ```bash
    npm run build
    ```

## Project Structure
- `src/components/`: Individual UI components (Hero, Services, etc.)
- `src/App.jsx`: Main layout assembly.
- `src/index.css`: Global styles and Tailwind configuration.
